<Info>
  **Prerequisite** You should have a frontend application and a backend
  application each.
</Info>

Step 1. Install Ploton Backend SDK in your Backend Server or Service:

<CodeGroup>

```bash npm
npm install @ploton/backend-sdk
```

```bash golang
go get https://github.com/plotondev/ploton/backend-sdk
```

</CodeGroup>

Step 2. Install Ploton Frontend SDK in your Frontend application:

```bash npm
npm install @ploton/frontend-sdk
```

Step 3. Authenticate with Ploton and get your tokens

With a simple call to ploton sever, you can get `tokens` that your frontend and users would require.

<CodeGroup>

```bash typescript
const { token } = ploton.authenticate(user, accessToken, hash);
```

```bash golang
token := ploton.authenticate(user, accessToken, hash)
```

</CodeGroup>

Step 3. Setup frontend

<Tip>You can setup fronend first too.</Tip>

<Frame>
  <Ploton user={user} token={accessToken} hash={hash} />
</Frame>

## Troubleshooting

Here's how to solve some common problems when working with Ploton SDKs.

<AccordionGroup>
  <Accordion title="App not registered">
    See if your key or subscription has expired. If so, get a new one.
  </Accordion>
  <Accordion title="No such endpoint.">Do not use raw endpoints.</Accordion>
  <Accordion title="Getting an unknown error">
    Try reaching out to support.
  </Accordion>
</AccordionGroup>
